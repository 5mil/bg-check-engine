Access control

This GitHub Pages frontend uses GitHub OAuth in the browser and stores the access token locally. For production, move OAuth token exchange to a backend or serverless function and restrict repo/org membership there.

  • Client ID is configured in config.js.
  • Optional org/user allowlist is enforced in the frontend.
  • API base URL is configured in config.js.
